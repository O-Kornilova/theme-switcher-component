## Getting Started

### 1. Clone the repository
```bash
git clone https://your-repo-url.git
cd womapp-header
```

### 2. Install dependencies
```bash
npm install
# or
yarn
```

### 3. Start the development server
```bash
npm run dev
# or
yarn dev
```

### 4. Build for production
```bash
npm run build
# or
yarn build
```

### 5. Run tests
```bash
npm run test
# or
yarn test
```
